# Technical Writing Portfolio – Orge Castellano 

This repository contains selected samples of technical documentation designed to demonstrate my approach to product-focused, developer-oriented documentation.

## Focus areas
- API documentation
- Troubleshooting and user guidance
- Feature documentation aligned with product workflows
- Docs-as-code using Markdown and GitHub

## Approach
I focus on:
- reducing user friction
- explaining systems, not just interfaces
- writing for real-world use cases and edge cases
- keeping documentation structured, concise, and actionable
